# enspace-documentation
A Docusaurus-based hosting for the markdown documentation of all enspace IT projects.

## Installation
```bash
npm install
npm --version
npx create-docusaurus@latest enspace-documentation classic
```

## Local Development
```bash
npm run start
```
Starts a local development server with live reloading.

## Build
```bash
npm run build
```
Generates static content into the `build` directory.
